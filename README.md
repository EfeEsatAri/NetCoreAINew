# 🤖 AI Masterclass: C# ile 20 Yapay Zeka Entegrasyonu – Part 2

Bu repository, Udemy üzerinde yer alan “AI Masterclass: C# ile 20 Yapay Zeka Entegrasyonu – Part 2” eğitimini tamamlamamın ardından geliştirdiğim projeleri içermektedir.

Bu eğitim kapsamında, C# ve .NET teknolojileri kullanılarak farklı yapay zeka servislerinin uygulamalara entegrasyonu pratiğe dökülmüştür. Gerçek dünya senaryoları üzerinden API kullanımı, veri işleme ve AI servis entegrasyonları gerçekleştirilmiştir.


# 🚀 Proje İçeriği

Bu repository içerisinde:

🔹 OpenAI API entegrasyonu

🔹 Metin üretme (Text Generation)

🔹 Duygu analizi (Sentiment Analysis)

🔹 Görsel üretme (Text-to-Image)

🔹 Görsel analiz işlemleri

🔹 PDF içerik okuma ve AI ile yorumlama

🔹 JSON veri işleme

🔹 HTTP Client ile API istekleri

🔹 .NET Console projeleri ile AI uygulamaları

gibi toplamda 20 farklı yapay zeka entegrasyon örneği bulunmaktadır.


# 🛠️ Kullanılan Teknolojiler

✅ C#

✅ .NET

✅ REST API

✅ JSON

✅ HttpClient

✅ OpenAI API

✅ Hugging Face API


# 🎯 Amaç

Bu proje ile amaçlanan:

Yapay zeka servislerini C# ile entegre etmeyi öğrenmek

API kullanım mantığını kavramak

Gerçek dünya senaryolarında AI çözümleri geliştirmek

.NET projelerinde modern AI servislerini kullanabilmek


# 📌 Not

Bu proje eğitim amaçlı geliştirilmiştir. API anahtarları güvenlik nedeniyle paylaşılmamıştır. Kendi API anahtarınızı ekleyerek projeleri çalıştırabilirsiniz.
